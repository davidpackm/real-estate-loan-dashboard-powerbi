# Dashboard Power BI de pilotage des prêts immobiliers

## Présentation

Ce projet consiste en la conception d’un dashboard Power BI destiné au pilotage de l’activité des prêts immobiliers des agences de **Crédit Breton**, dans le cadre d’un **cas simulé**.

L’objectif du dashboard est de centraliser les principales informations liées aux demandes de prêts, de comparer les performances des agences du réseau et de faciliter la consultation des indicateurs clients utiles à l’analyse des dossiers.

## Contexte métier

Dans ce cas d’usage, le suivi de l’activité des prêts immobiliers implique de croiser un volume important d’informations relatives aux demandes, aux agences et aux clients. Sans outil de pilotage unifié, cette organisation rend l’analyse plus lente, complexifie le suivi des performances et limite la visibilité globale sur l’activité.

Ce dashboard a donc été conçu comme un outil d’aide au pilotage permettant de :
- suivre l’évolution des demandes de prêts immobiliers ;
- analyser les taux d’acceptation ;
- comparer les performances des agences ;
- accéder rapidement aux informations clés des clients.

## Objectifs du projet

Ce projet répond à plusieurs objectifs :
- fournir une vue d’ensemble claire de l’activité des prêts immobiliers ;
- suivre les principales tendances liées aux demandes de prêts ;
- comparer les performances des agences du réseau ;
- synthétiser les indicateurs clients utiles à l’analyse ;
- faciliter la recherche et la consultation de la situation d’un client.

## Structure du dashboard

Le dashboard est organisé en **quatre pages principales**.

### 1. Les demandes de prêts

Cette page permet de suivre les principaux indicateurs liés à l’activité de prêt immobilier :
- historique des demandes ;
- évolution du taux des prêts acceptés ;
- évolution du montant moyen des opérations ;
- dossiers en cours de traitement.


![Page 2 : Demande de prêts](https://github.com/davidpackm/real-estate-loan-dashboard-powerbi/blob/e44701188018a42e443defe1f8891c584538114d/credit-breton-p2.png)


### 2. Les performances des agences

Cette page permet de comparer les agences entre elles à partir de plusieurs indicateurs de performance :
- localisation des agences du réseau Crédit Breton ;
- nombre de demandes reçues ;
- classement des agences par volume de demandes ;
- taux d’acceptation des prêts par agence.

![Page 3 : Performance des agences](https://github.com/davidpackm/real-estate-loan-dashboard-powerbi/blob/e44701188018a42e443defe1f8891c584538114d/credit-breton-p3.png)


### 3. Les indicateurs clients

Cette page présente une synthèse des principaux indicateurs clients utiles à l’analyse des dossiers de prêts immobiliers.

### 4. La liste des clients

Cette page permet d’afficher, filtrer et rechercher la situation d’un client en particulier afin d’accéder rapidement à ses informations.

## Valeur analytique

Le dashboard a été conçu pour apporter plusieurs niveaux de lecture :
- une vision globale de l’activité ;
- une analyse comparative des performances des agences ;
- une synthèse des indicateurs clients ;
- une consultation ciblée à l’échelle individuelle.

Il permet ainsi de mieux suivre la dynamique des demandes, d’identifier les écarts de performance entre agences et de faciliter l’accès à l’information client dans une logique de pilotage opérationnel.

## Outils utilisés

- **Power BI** pour la conception du dashboard et la visualisation des données ;
- **Power Query** pour la préparation et la transformation des données ;
- **DAX** pour la création des mesures et indicateurs.

## Compétences mobilisées

- analyse d’un besoin métier ;
- structuration d’un dashboard de pilotage ;
- conception de KPI ;
- comparaison de performances entre entités ;
- synthèse d’indicateurs clients ;
- data visualisation avec Power BI.

## Données

Les données utilisées dans ce projet relèvent d’un **cas simulé à finalité pédagogique**.
